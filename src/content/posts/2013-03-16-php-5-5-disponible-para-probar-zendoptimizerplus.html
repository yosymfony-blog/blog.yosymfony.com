--- author: victor excerpt: '' categories: - PHP tags: - optimizerplus - 'php 5.5' permalink: /php-5-5-disponible-para-probar-zendoptimizerplus no_html_extension: true layout: post title: 'PHP 5.5: Disponible para probar ZendOptimizerPlus' --- El pasado 28 de Febrero anunciamos que PHP 5.5 vendría con las pilas cargadas en cuanto a rendimiento porque incluiría de serie ZendOptimizerPlus, un opcode-cache que hará que la ejecución de script se realice con mayor velocidad. Esta mejoría se debe a que la votación del RFC terminó a favor de incluirlo desde la próxima versión de PHP. Pues bien... ayer Rasmus Lerdorf, creador de PHP, publicó un twit indicando que está disponible para probar el ZendOptimizerPlus:
You can start testing today with the new 5.5 cache by installing pecl.php.net/package/ZendOp… for PHP 5.2-5.4 — Rasmus Lerdorf (@rasmus) 15 de marzo de 2013
Solo queda descargarlo (requiere PHP >= 5.2) y probar a ver que tal va...
