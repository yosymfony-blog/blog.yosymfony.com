--- author: victor excerpt: '' categories: - PHP - Symfony tags: - libros - micro-framework - silex permalink: /manual-de-silex-en-espanol no_html_extension: true layout: post title: 'Manual de Silex en español' --- Silex es un microframework desarrollado por Sensio Labs, los creadores de Symfony, destinado a la pequeña y mediana aplicación. Aunque Symfony es uno de los mejores web framework para desarrollar grandes aplicaciones de forma ágil y siguiendo buenas prácticas, no parece muy lógico emplearlo en pequeños proyecto que apenas usan base de datos. El concepto sobre el que pivota Silex es el de microframework, es decir, que nuestra solución se encuentre entera contenida en un solo archivo: 
// index.php

require_once __DIR__.'/../vendor/autoload.php';

$app = new Silex\Application();

$app->get('/usuario/{name}', function ($name) use ($app) {
    return 'Bienvenido '.$app->escape($name);
});

$app->run();
Este es un ejemplo sencillo que contiene definida una ruta y la acción asociada a ella. Para probarlo, basta con acceder a /usuario/yoSymfony y verás que responde "Bienvenido yoSymfony". Por supuesto, requiere tener Silex instalado tal como explica Javier Eguiluz. Silex tiene soporte para realizar pruebas funcionales mediante el módulo HttpKernel de Symfony2 que incluye.
